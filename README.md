# ScrambleFarm

## 概要
ScrambleFarmは、作物育成とローグライクアクションを組み合わせたUnity製2Dアクションゲームです。

プレイヤーはダンジョン内で敵と戦いながら作物を育て、収穫した作物を食べることでステータスを強化します。  
限られた時間の中で生存し、より高いスコアを目指すゲームです。

## 制作目的
ゲームクリエイター職への就職活動に向けて、企画・実装・改善の過程を記録するために制作しています。

完成品だけでなく、開発中の課題、修正内容、設計の工夫もGitHub上に残すことで、自分の制作過程を見せられるようにすることを目的としています。

## 使用技術
- Unity
- C#
- GitHub
- GitHub Desktop

## ゲームの特徴
- 作物を育ててステータスを強化するシステム
- 戦闘と農業を組み合わせたゲームサイクル
- 複数の武器を切り替えて戦うアクション要素
- 15分間の生存を目指すローグライク形式
- スコアアタック要素

## 操作方法

| 操作 | 内容 |
|---|---|
| W / A / S / D | 移動 |
| 左クリック | 攻撃 |
| 右クリック | 種を植える |
| F | 水やり |
| X | メニューを開く |
| マウスホイール | 種の切り替え |
