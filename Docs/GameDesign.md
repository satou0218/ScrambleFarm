# ScrambleFarm 企画書

## 1. ゲーム概要
ScrambleFarmは、作物育成とローグライクアクションを組み合わせた2Dアクションゲームです。
プレイヤーは制限時間内に敵と戦いながら作物を育て、収穫した作物を食べたり売ったりすることでステータスやスキルを強化し、生存と高スコアを目指します。

## 2. コンセプト
「農業で強くなり、戦闘で農業を支える」
作物を育てるだけでなく、敵を倒すことで畑の栄養を回復させるなど、農業要素と戦闘要素が互いに関係する構造を目指しています。

## 3. 基本ゲームサイクル
1. フィールドを移動する
2. 畑に種を植える
3. 水やりをする
4. 敵と戦う
5. 作物を育てる
6. 作物を収穫する
7. 作物を食べる・売る
8. ステータスやスキルを強化する
9. より長く生存し、高スコアを目指す

## 4. 操作方法

| 操作 | 内容 |
|---|---|
| W / A / S / D | 移動 |
| 左クリック | 攻撃 |
| 右クリック | 種を植える |
| F | 水やり |
| X | メニュー |
| マウスホイール | 種の切り替え |

## 5. 作物システム
作物はStage0〜Stage3まで成長します。

成長には時間、水分、栄養が関係します。

- 水分は水やりで回復する
- 栄養は敵を畑の上で倒すことで回復する
- 作物ごとに対応するステータスが異なる

## 6. 戦闘システム
プレイヤーは複数の武器を切り替えながら敵と戦います。

実装予定の武器は以下です。

- 剣
- 槍
- ハンマー
- 杖
- 魔導書
- グローブ

武器ごとに攻撃力、攻撃範囲、クールタイムを変える予定です。

## 7. スキルシステム
作物の売買やボス撃破によってスキルを獲得し、プレイヤーを強化する仕組みを予定しています。

通常スキルと、ボス撃破で得られる災害スキルを分けて実装する予定です。
